# Decentralized Identifiers (DIDs) v1.0

Source: https://www.w3.org/TR/did/
Retrieved: 2026-02-06
Status: W3C Recommendation (2022-07-19)

Notes:
- Defines decentralized identifiers (DIDs) as URIs and specifies their syntax, data model, and representations.
- Defines DID documents and how they express verification methods and services.
- Specifies DID resolution and dereferencing concepts and related metadata.
- Explains the DID method concept and how method-specific rules are defined outside the core spec.
