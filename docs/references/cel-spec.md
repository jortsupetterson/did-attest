# Cryptographic Event Log (CEL) v0.1

Source: https://w3c-ccg.github.io/cel-spec/
Retrieved: 2026-02-06
Status: Draft (W3C CCG)

Notes:
- Defines cryptographic event logs and algorithms for creating, updating, deactivating, and verifying them.
- Event log entries are linked by hashes and can include witness signatures.
- Provides a foundation for verifiable history of state changes across different applications.
